## ğŸš€ Getting Started

Follow these steps to run the project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Laksh-Devloper/Portfolio-2.git
    cd Portfolio-2
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
